def toNum(dataf,column):
    lst = dataf[column].unique()
    dataf = dataf.replace(to_replace = lst, value= list(range(0,len(lst))))
    return dataf

#Find the optimal k using for loop
error_rate = []
# Might take some time
for i in range(1,40):
    
    knn = KNeighborsClassifier(n_neighbors=i)

Seems like Our models are really bad at predicting employment based on the demographics given. However, random forrest seems to be able to predict our data a lot more accurately based on it's AUC score. Random forrest seems to be the best classifier out of those identified, while SVM seems to be the worst classifier for our data.
